# Archiwum

Pliki przeniesione w ramach porządkowania projektu (luty 2026).

## Zawartość

- **snippets/** – stare wygenerowane snippet HTML z testów (wanilia.pl, fizjoterapia2-0.pl)
- **dist/** – stare pliki testowe (final_test.html, short.html, wanilia_v3.html itd.)
- **snippet.html, test_snippet.html, wanilia_snippet.html, test_backward.html** – stare outputy testowe
- **wzor.txt** – wycinek pluginu WP (fizjoterapia2-0.pl) jako odniesienie formatu
- **config_sample.json** – stary format konfiguracji (JSON); aktualnie: config.yaml
- **README_ai_buttons.md** – dokumentacja v2.0 (przestarzała); aktualnie: README.md
